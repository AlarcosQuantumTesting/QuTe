QuTe: Quantum Testing

Design, simulate, and verify quantum circuit behavior

[ OVERVIEW ]

QuTe is an advanced engineering framework for designing, simulating, and verifying quantum circuit test suites. You can specify deterministic test cases to confirm exact state transitions, or build stochastic assertions to validate state probability distributions.

Deterministic test cases define a classical input bitstring and the expected output state after the circuit runs, letting you confirm gate correctness step by step. Stochastic test cases let you declare the probability that each basis state should appear in measurement results, making it possible to verify superposition and entanglement behaviour against precise statistical thresholds.
